FAQ

            

                
Qu'est-ce que le langage C ?

                
Le C est un langage de programmation généraliste créé en 1972. Il est utilisé pour les systèmes d'exploitation, les applications embarquées et bien plus.

            

            

                
Comment démarrer en C ?

                
Installez un compilateur comme GCC, utilisez un IDE comme Code::Blocks, et commencez avec un tutoriel de base.

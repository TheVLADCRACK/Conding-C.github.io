Introduction

Le langage C est l'un des langages les plus puissants et performants, largement utilisé pour le développement système, les applications embarquées et bien plus.

Pointeurs

Les pointeurs sont des variables qui stockent l'adresse mémoire d'une autre variable. Ils sont essentiels pour manipuler la mémoire de manière efficace.


int x = 42;
int *ptr = &x;
printf("Valeur de x : %d", *ptr);

Gestion de Fichiers

En C, la gestion des fichiers se fait via des fonctions comme fopen, fclose, fprintf, etc.


FILE *file = fopen("example.txt", "w");
if (file != NULL) {
    fprintf(file, "Bonjour, C !");
    fclose(file);
}

Multi-threading

Le multi-threading permet d'exécuter plusieurs tâches simultanément en utilisant des bibliothèques comme pthread.


#include 
void *routine() {
    printf("Thread en cours d'exécution.");
}
pthread_t thread;
pthread_create(&thread, NULL, routine, NULL);
pthread_join(thread, NULL);
